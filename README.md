# The First Letter Method

A simple web tool for memorizing text using the first letter technique. Paste any text — a poem, speech, definition, lyrics — and it hides everything except the first letter of each word. Reveal words or lines one at a time to test your recall.


Live at: https://dobrosketchkun.github.io/the_first_letter_method/


## How it works

1. Paste the text you want to memorize.
2. Hit "Start Studying." Each word collapses to its first letter.
3. Click any word to reveal it. Click the arrow on a line to reveal the whole line.
4. Use "Reveal All" / "Hide All" to toggle everything at once.

## Features

- Per-word and per-line reveal/hide
- Adjustable text size (A+/A-)
- 9 themes: Parchment, Clean Light, Dark, Midnight, Ocean, Forest, Rose, High Contrast, Dyslexia Friendly
- Theme preference saved across sessions
- Single HTML file, no dependencies, no build step
